# Task 6: Sales Trend Analysis Using Aggregations

## Objective
Analyzed monthly revenue and order volume using SQL aggregations.

## Tool Used
- MySQL Workbench

## Dataset
- online_sales (order_date, amount, product_id)

## Analysis Performed
- Grouped data by year and month
- Calculated monthly revenue using SUM()
- Calculated monthly order volume using COUNT(DISTINCT)
- Sorted results using ORDER BY
- Identified top 3 months by revenue

## Outcome
Learned how to group data and analyze time trends using SQL.
